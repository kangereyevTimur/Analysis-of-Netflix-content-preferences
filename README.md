# Analysis of Netflix Content Preferences

## Description
This project analyzes Netflix movies and TV shows to understand viewer preferences across genres and countries.  
We explore the dynamics of content releases, genre distribution, and clustering by countries and genres.

## Dataset
Netflix Movies and TV Shows Dataset  
- Source: [Kaggle Netflix Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)  
- Includes information about titles, release year, type (Movie/TV Show), country, genres, and more.

## Project Steps
1. **Exploratory Data Analysis (EDA)**  
   - Count of movies and TV shows by type  
   - Number of releases per year  
   - Top countries by content volume  
   - Top genres and their distribution  

2. **Visualization**  
   - Trend of content releases over time  
   - Genre distribution plots  
   - Interactive visualizations with Plotly  

3. **Clustering Analysis**  
   - Genre-based clustering using KMeans  
   - Regional clustering of content  

4. **Comparative Analysis**  
   - Activity of content across top countries  
   - Yearly trends in releases by country
